사용 가이드
  • 데이터 초기화: 최초 1회만 실행하면 됩니다. 데이터가 이미 적재되어 있으면 버튼을 눌러도 중복 적재되지 않습니다.
  • Fold 선택: Fold 1-7은 Training Set, Fold 8은 Test Set입니다. 먼저 Training Set으로 전략을 검증하세요.
  • 필터 컬럼: confidence는 모델 확신도(0~0.5), pred_proba_up은 상승 확률(0~1)입니다.
  • 필터 모드: 고정값은 절대값 기준, 백분위수는 데이터 분포의 상위 N%를 선택합니다.
  • 포지션 크기: 비워두면 Kelly Criterion이 자동 계산됩니다. 리스크를 줄이려면 고정 비율(예: 50%)을 입력하세요.
